# mixed-effect-nested-statistical-analysis

## <p align="center"><b>General</b></p>

This repository provides R scripts to model mixed effect nested statistical analysis done on the zone-specific morphology. zone is the fixed effect, subject is the random effect, and vertebra is the random effect nested in subject. 

It performs the statistical analysis, computes the power of the current analysis, and suggests the correct number of subjects for a full-scale analysis of a power 80% (in case you are currently doing a pilot study)

for replication, please change the directory to match yours and fill in the excel files with your data
